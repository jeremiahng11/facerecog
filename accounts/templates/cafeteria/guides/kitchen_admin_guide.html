Kitchen Admin Guide

Role: Kitchen Admin
Access: Menu management, Event Menus, Events, Counter Displays
Last Updated: April 2026

1. Your Role

As a Kitchen Admin, you manage the daily menu, event catering menus, and can access counter displays. You do not have access to user management, credit adjustments, or system settings — those are Administrator-only.

2. Navigation

Your top navigation bar has:

  • Menus — manage daily menu items
  • Event Menus — create/edit catering packages
  • Events — view upcoming approved event bookings
  • Counter — access kitchen counter displays

3. Daily Menu Management

3.1 Adding Menu Items

  1. Go to Menus
  2. Click Add Item
  3. Fill in: Name, Description, Staff Price, Public Price
  4. Select Menu Type: Local Kitchen, International Kitchen, or Cafe Bar
  5. Set Category (e.g. Rice Meals, Noodles, Hot Drinks)
  6. Tick Vegetarian for plant-based items (shows a green badge to users)
  7. Set Daily Quantity — stock count for the day
  8. Upload a Photo (optional but recommended)
  9. Click Save

3.2 Daily Stock

  • Each morning, update the day's stock quantities from the Menus page
  • As orders come in, remaining stock decreases automatically
  • When stock hits 0, the item shows as "Sold Out"
  • You can also toggle Available to temporarily hide an item

3.3 Editing / Removing Items

  • Click any item to edit its details, price, or photo
  • Toggle Available off to hide without deleting

4. Event Menus (Catering Packages)

  1. Go to Event Menus
  2. Click Add Menu to create a new package
  3. Set: Package Name, Description, Price Per Pax, Min/Max Pax
  4. Add Menu Components: Appetiser, Main, Side, Drink, Dessert, Other
  5. Each component has a name and optional description
  6. Staff will see these packages when booking events

5. Events

  • Go to Events to see upcoming approved event bookings
  • View details: date, time, venue, pax count, menu selection, special requests
  • Use this to plan kitchen preparation for catering events

6. Counter Displays

  • Go to Counter to access the Displays Hub
  • Select your counter: Local Kitchen, International Kitchen, or Cafe Bar
  • The counter display shows incoming orders with item lists
  • Two tabs: Active (current orders) and No Show (uncollected after 20+ minutes)
  • Use the QR scanner to validate collection when staff pick up their food
Tip: Keep the counter display open on your kitchen tablet. It auto-refreshes every 20 seconds to show new orders.
