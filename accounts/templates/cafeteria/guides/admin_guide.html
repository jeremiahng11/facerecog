Administrator Guide

System: FaceID Portal — Cafeteria Management System
Role: Administrator
Last Updated: April 2026

1. Dashboard Overview

After logging in, you land on the Admin Dashboard showing:

  • Today's KPIs — Revenue, orders, credits used, refunds
  • 7-day revenue chart
  • Top 5 items sold this week
  • Recent orders list

Quick-access buttons: Displays, Menu, Stock, Orders, Reports, Staff, Events, Event Menus, Settings.

2. Managing Users

2.1 Add a Single User

  1. Go to Users → Add User
  2. Fill in: Staff ID, Email, Full Name, Department, Password
  3. Select Employment Type: Permanent, Temp Staff, or Intern
  4. For Temp/Intern: set the Last Day — account auto-disables after this date
  5. Set Monthly Credit (default S$50)
  6. Prorate Credit checkbox (checked by default) — auto-calculates opening balance based on remaining days this month
  7. Uncheck to enter a manual credit amount
  8. Click Create User

2.2 Bulk Import

  1. Go to Users → Bulk Import
  2. Download the CSV Template or Excel Template
  3. Fill in user data (staff_id, email, full_name, password are required)
  4. Optional columns: department, staff_type (permanent/temp/intern), contract_end_date (YYYY-MM-DD)
  5. Upload the file — credits are auto-prorated

2.3 Edit / Deactivate Users

  • Click any user in the Users list to edit their details
  • Toggle Active to deactivate (blocks login)
  • Change Staff Type and contract end date for existing users
  • Adjust credit balance from the Staff Credits page

3. Menu Management

  1. Go to Menu from the dashboard
  2. Add Item: name, description, staff price, public price, menu type (Local/International/Cafe Bar), category, photo
  3. Tick Vegetarian for plant-based items
  4. Set Daily Quantity and Low Stock Threshold
  5. Toggle Available to hide items without deleting

4. Daily Stock Management

  1. Go to Stock each morning
  2. Set the day's quantities for each item
  3. As orders come in, remaining stock decreases automatically
  4. Items reaching 0 show as "Sold Out" in all ordering interfaces

5. Orders

  • Orders — view all orders, filter by status
  • Cancel an active order: credits are auto-refunded, stock is reinstated
  • My Orders — place orders as admin (e.g. for visitors)

6. Reports

Go to Reports — filter by Today / Week / Month:

  • Revenue KPIs — total revenue, credits used, refunds, net
  • By Menu Type — Local Kitchen, International Kitchen, Cafe Bar, Mixed
  • By Payment Method — Staff Credits, Stripe, PayNow, Cash, Terminal
  • Daily Breakdown table
  • Vending Machines — total payable to vendor, per-machine breakdown, link to full vending report with CSV download
  • Refunds & No Shows — cancelled orders and auto-cancelled no-show orders

7. Settings

7.1 Kiosk Timeouts

  • Idle landing countdown — seconds before kiosk resets to idle screen
  • Session idle timeout — auto-logout after inactivity
  • Post-print auto-return — seconds after print before returning to idle

7.2 Credit Settings

  • Working Days / Month — used to prorate credit for new staff (e.g. 30 days → joining on 15th = 50% credit)

7.3 Auto-Cancellation

  • Kitchen Cutoff Time (default 17:01) — uncollected kitchen orders auto-cancel after this time daily
  • Cafe Bar Cutoff Time (default 20:01) — uncollected cafe bar orders auto-cancel daily
  • No Show Timer (default 20 min) — orders not collected within this time appear in the No Show tab at counters
  • Credits are automatically refunded for cancelled orders
  • For mixed orders with partial collection, only uncollected items are refunded

7.4 Operating Hours

  • Set opening/closing times for Kitchen and Cafe Bar
  • Toggle per weekday (Mon–Sun)
  • Add public holidays to close specific counters

8. Events & Catering

  1. Go to Events from the dashboard
  2. View all event bookings (pending, approved, rejected, cancelled)
  3. Approve or reject staff event requests
  4. Book an Event — admin bookings are auto-approved
  5. Event Menus — create/edit catering packages with components (mains, sides, drinks, desserts), price per pax, min/max pax

9. Vending Machine Integration

  • Vending API docs: Settings → Vending API (or download .doc)
  • Set VENDING_API_KEY environment variable on Railway
  • Vending machines deduct staff credit via QR scan
  • Track transactions in Reports → Vending Machines
  • Download monthly CSV from Reports → Full Vending Report for vendor reconciliation

10. Monthly Credit Reset

  • Credits reset automatically on the 1st of each month via GitHub Actions cron
  • Any uncollected orders from the previous day are refunded before the reset
  • Expired temp/intern accounts are auto-disabled

11. Face ID Management

  • Staff enroll Face ID from their Profile page (live camera only — no photo upload)
  • Admin can toggle Face Enabled per user in the edit user page
  • View login attempts in Users → Face Logs
  • Liveness detection prevents photo spoofing
